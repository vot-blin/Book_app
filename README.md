# Bookhouse - «Минималистичное управление персональной библиотекой»

Приложение — это цифровой аналог личной книжной полки и читательского дневника. Оно помогает: Не забыть о книгах, которые хочется прочитать. Сосредоточиться на текущем чтении, отслеживая прогресс. Организовать свою прочитанную литературу так, как удобно пользователю. Иметь быстрый доступ ко всей своей библиотеке в одном месте.
е
Цель приложения: Предоставить пользователю простой и интуитивно понятный инструмент для учета прочитанных книг, планирования чтения и формирования своей цифровой библиотеки. 

## Основные сущности и их атрибуты:

*Книга*

Обязательные атрибуты:
- Название (обязательное)  
- Автор (обязательное)  
- Статус чтения:  
  - Не начата 
  - В процессе  
  - Прочитана  
  - Отложена  
- Дата добавления (автоматически устанавливается при создании записи)  
- Краткое описание / аннотация (необязательно)
- Обложка книги (изображение или URL)

*Коллекция*

Виртуальные "полки" для организации книг.
Примеры: "Хочу прочитать", "Читаю сейчас", "Прочитано", "Отложено", "Любимое", а также пользовательские коллекции ("Фэнтези", "Для учебы", "К отпуску").

*Прогресс чтения*
Атрибуты: Текущая страница (или процент), общее количество страниц, дата начала чтения, дата завершения (для прочитанных книг), заметки.
Функционал: Возможность вручную обновлять прогресс.

*Читатель*
Личные данные минимальны (логин/email, пароль).

#### **3. Основные операции (функциональные требования)**

Пользователь может:
- **Добавлять** новую книгу, указав минимум — название и автора.  
- **Просматривать** список всех книг с возможностью фильтрации по статусу.  
- **Редактировать** информацию о книге (включая изменение статуса).  
- **Удалять** книгу из библиотеки.  
- **Помечать статус** книги (например, переключить с «Не начата» на «В процессе»).  

Дополнительно (опционально, но в рамках минимализма):
- **Поиск по названию или автору** (без сложных фильтров).  
- **Сортировка** — по названию, автору или дате добавления.

## Основные процессы (Use Cases / Сценарии):

Управление библиотекой:
- Добавление новой книги (ручной ввод, сканирование ISBN, импорт из файла).
- Редактирование информации о книге (исправление опечаток, добавление своих тегов).
- Удаление книги из библиотеки.
- Перемещение книги между коллекциями ("полками").

Процесс чтения:
- Начало чтения книги (перевод книги в статус "Читаю", установка общего количества страниц).
- Обновление прогресса (ввод текущей страницы).
- Добавление приватной заметки/закладки к определенной странице или главе.
- Завершение чтения (отметка как "Прочитано", фиксация даты завершения).

Обзор и поиск:
- Просмотр библиотеки в разных режимах (списком, сеткой, отсортировано по названию, автору, дате добавления).
- Фильтрация книг по статусу ("Хочу прочитать", "Прочитано").
- Поиск книги по названию или автору.
- Быстрый доступ к "текущей книге" (книгам в статусе "Читаю").
